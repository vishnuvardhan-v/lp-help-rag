Help¶

To access the User Guide, Release Notes or Support, click on the Help in the top menu bar

User Guide¶

Select User Guide to access an online version of the User Guide

Support¶

Select Support to go to the Cambium Networks Support Center.

Release Notes¶

Select Release Notes to see the latest Release Notes for the current version and the history of changes in previous releases.

About¶

Select About to see background information on LINKPlanner and for links to contact the LINKPlanner team or the Cambium Community Forum

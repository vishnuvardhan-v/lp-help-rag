Using LINKPlannerÂ¶

LINKPlanner allows the planner to design PMP networks.

To achieve this goal, follow this process:

  1. Start the application and set options. See Starting the Application.

  2. Build a project to model a PMP network See Projects.

  3. Enter details of all sites in the project. See Sites.

  4. Plan a PMP network, see Planning PMP networks

  5. View the Bill of Materials for the project. See Bill of Materials for Project.

  6. Export and report project, network, site and link data. See Exporting and Reporting

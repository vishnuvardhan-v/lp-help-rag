Starting the Application¶

To start LINKPlanner go to https://lp.cambiumnetworks.com

To access the Terms & Conditions at any time, click on Terms & Conditions at the bottom of this page.

Click on Sign In to access the Cambium Single Sign In page, this uses the same Cambium Single Sign On (SSO) process as cnMaestro and our Learning Management System. For more information on the Multi-Factor Authentication (MFA), see https://community.cambiumnetworks.com/t/multi-factor-authentication/77747.

On first login you will see the Terms and Conditions screen and must accept the Terms and Conditions before using LINKPlanner

When signed in the Getting started screen for new users is displayed.

Before using the application, set preferences. See Preferences.

To start using LINKPlanner, see Projects

PMP Network Design Process¶

This section provides a quick process to design a PMP network. It describes how to build a PMP network with multiple network sites, Network Devices and Subscriber sites in a few simple steps.

Before starting this process, complete the following steps:

  1. Login to LINKPlanner, see Starting the Application.

  2. Enter the correct details in the Preferences page. See Preferences.

Build the network in four simple steps:

  1. Add network sites from csv. See Importing Sites.

  2. Add subscriber sites from csv. See Importing Sites.

  3. Add network devices. See PMP Network Devices.

  4. Connect subscribers using Best Server Analysis. See Best Server Analysis.

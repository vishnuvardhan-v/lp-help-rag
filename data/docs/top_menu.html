Top Menu¶

The top menu provides quick access to key functions, including Save, Discard, Reports, Import, Export, Preferences and Help.

Projects¶

Under the Projects menu, find the options to import an existing LINKPlanner project, create a new project, save the current project, or discard changes. See Projects.

Reports¶

Different reports can be generated for the projects. To generate project reports click Reports menu. See Reports.

Import¶

To import Network Sites and Subscriber Sites from a CSV or KML file click Import menu. See Importing Sites.

Export¶

To export the current project as a LINKPlanner project file or a Google Earth file click Export menu. See Export.

Preferences¶

To set the user account preferences click Preferences menu. See Preferences.

Help¶

To access the User Guide, Release Notes or Support, click on the Help menu. See Help.
